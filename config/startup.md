# 开机自启动功能记录

## 背景

用户希望 learn++ 可以跟随 Windows 开机自动启动，减少每天手动打开客户端的操作。

## 实现思路

1. 在设置页“登录设置”区域新增“开机自启动”开关。
2. 主进程通过 Electron 的 `app.setLoginItemSettings()` 写入系统登录项。
3. 设置页读取配置时同步 `app.getLoginItemSettings().openAtLogin`，保证显示的是系统真实状态。
4. 该能力只作为本地客户端行为，不影响网络学堂登录态。

## 当前结果

用户可以在设置页直接开启或关闭开机自启动。Windows 下会使用当前应用可执行文件路径作为启动目标。
