# Learn++ 2.0 UI 更新框架交接

本文档总结本轮对话中完成的 Learn++ 2.0 UI 大体框架，供后续 agent 在实现细节功能时保持视觉和信息架构一致。

## 1. 产品定位

- 主产品名称为 `Learn++ 2.0`。
- `甘蔗 Tutor` 是 Learn++ 内置 AI 学习助手功能，不是独立产品或联名产品。
- 全局只保留一个 AI 角色：甘蔗 Tutor。
- 不再出现 `Learn++ × 甘蔗 Tutor`、`AI伙伴`、`随便` 或第二个 AI 角色。
- 整体方向是清爽、现代、柔和、圆润、可爱但专业的清华网络学堂桌面客户端。

## 2. 视觉体系

- 主色：紫色品牌色，近似 `#6B46C1` / `#4C1D95`。
- 辅助色：甘蔗绿，近似 `#7CB342` / `#4F8A10`，仅用于甘蔗 Tutor 相关入口。
- 背景：主界面统一纯净白色，登录页保留浅紫白渐变与柔和装饰。
- 卡片：白色背景、淡紫边框、柔和紫色阴影、较大圆角。
- 按钮：
  - 主按钮使用紫色渐变。
  - Tutor 辅助按钮使用浅绿色底和绿色文字。
  - 学术诚信风险提示使用浅黄色，不使用刺眼红色。
- 所有图片禁止作为拖拽热区，图片本身不可被鼠标拖拽。

## 3. 窗口与全局外壳

- Electron 主窗口和登录窗口均使用自绘圆角矩形界面。
- 系统传统标题栏已移除，使用自绘右上角窗口控制按钮。
- 右上角按钮包括：最小化、最大化/还原、关闭。
- 关闭按钮带非线性淡出/缩放动画。
- 顶部可拖拽移动窗口，但交互元素、图片、按钮、输入框都必须为 `no-drag`。
- 主界面底色为纯白，避免脏灰或半透明底色。

相关文件：
- `src/renderer/src/components/WindowControls.tsx`
- `src/main/ipc/app.ts`
- `src/renderer/src/styles/tsinghua.css`

首页卡片框架：

## 2026-07-03 视觉整改第四批（作业页无界化 + 全局非线性动画）

### 作业页重设计（Homework.tsx）

问题：已完成作业仍显示红色截止警示（urgencyColor 对过期作业恒红）、行右侧大片空置、完成行整体 0.55 透明度发灰、lp2-card 边框卡片不符无界基调、每行重复塞甘蔗按钮、时间显示带秒。

改法：
- List.Item 卡片 → 无界网格行（lp2-hw-row：紧急度圆点 | 标题+元信息 | 右侧状态位）
- 右侧着陆点：已批阅显示大号紫色成绩（负分过滤铁律保留）+ 小字"已批阅"；已提交/已截止为浅色胶囊；未提交为"提交作业"主按钮
- 紧急色仅用于未提交（圆点+剩余天数），完成行时间为中性灰
- 时间格式 M月D日 HH:mm；描述压缩为单行 72 字
- 行内甘蔗按钮移除（工具栏已有）；分组标题降为小号灰字"未完成 · N"
- 作业/公告/讨论工具栏 + 课件工具栏统一无界填充控件

### 全局非线性动画系统（pass 6）

- 缓动令牌：--lp2-ease-out (0.22,1,0.36,1) 快出缓停；--lp2-ease-spring (0.34,1.45,0.64,1) 轻回弹
- 路由页面进入：.lp2-content > * 挂载时 8px 上浮淡入 0.3s
- 列表行级联进场：邮件/课件/作业/任务/课程进度/更新行，前 8 行错峰 25ms
- 交互微动效：全部按钮/导航项/chips/行 统一 transform+background 弹性过渡；按压 scale(0.96) 回缩；侧栏项 hover 右移 2px
- prefers-reduced-motion: reduce 时全部禁用
