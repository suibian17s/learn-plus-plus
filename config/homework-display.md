# 作业显示优化记录

## 背景

作业详情页存在三类显示问题：

- 截止时间、提交时间仍显示 ISO 原始字符串。
- 部分老师发布作业时附带的留言没有独立展示。
- 已批阅作业如果网络学堂没有返回具体分数，可能把异常值显示成 `-60分`。

## 修复思路

- 新增统一时间格式化工具，作业列表、详情页和甘蔗 tutor 作业扫描页共用。
- 从网络学堂作业对象中兼容读取老师留言、备注、说明等可能字段，并在详情页单独显示。
- 对成绩字段做保护：只接受有效非负数字作为分数；如果没有有效分数，则展示等级字段。

## 结果

- 作业截止时间显示为 `YYYY-MM-DD HH:mm:ss`。
- 有老师留言时，会在作业要求下方独立展示。
- 已批阅作业优先显示真实分数；没有分数但有等级时显示等级；异常负分不再展示。
